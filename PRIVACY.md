# Privacy Policy / 隐私说明

## English

FeignHelper runs locally inside IntelliJ IDEA.

- It does not collect analytics, telemetry, personal information, or project source code.
- It does not send project data to external services.
- It reads Java/Kotlin PSI indexes and Spring `application*` / `bootstrap*`
  configuration files only to resolve endpoint mappings.
- It writes an endpoint path to the system clipboard only when the user selects
  the copy action.
- Plugin settings are stored locally by the IntelliJ Platform.

Questions can be sent to `maybe.zhuo@qq.com`.

## 中文

FeignHelper 仅在 IntelliJ IDEA 本地运行。

- 不收集分析数据、遥测信息、个人信息或项目源代码。
- 不向外部服务发送项目数据。
- 仅为解析接口映射而读取 Java/Kotlin PSI 索引以及 Spring
  `application*` / `bootstrap*` 配置文件。
- 只有用户主动选择复制操作时，才会把接口路径写入系统剪贴板。
- 插件设置由 IntelliJ Platform 保存在本地。

如有隐私问题，请联系 `maybe.zhuo@qq.com`。
